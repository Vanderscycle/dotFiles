# [Nex-st] Nestjs

## Nest.js

To generate a new endpoint ( websocket, rest, graphql ) using the cli. You can use `--dry-run` to see changes.

```bash
nest g res project-api # --dry-run
```

For a simple addition:
```bash
nest g module postgres # --dry-run
```
